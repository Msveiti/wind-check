# Wind Checker

A brutalist, single-serving website that answers one question: **"Should I go outside?"** using real-time wind data from Open-Meteo.

### Features
- Geolocation with fallback to San Francisco
- Human-readable wind descriptions
- Compass arrow showing wind direction
- No API keys, no ads, no tracking
- Pure HTML/CSS/JS

### Live Site
[https://https://msveiti.github.io/wind-check/](https://https://msveiti.github.io/wind-check/)

### Why?
Because wind speed is the most underrated factor in whether a walk or bike ride is enjoyable. Most weather apps hide it behind icons. This puts it front and center.

### Local Development
Clone the repo and open `index.html` in a browser. No build step required.

### APIs Used
- [Open-Meteo](https://open-meteo.com/) (weather)
- [Nominatim](https://nominatim.org/) (reverse geocoding, optional)
